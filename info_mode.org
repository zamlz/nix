* Major mode for Browsing the Info Documentation Tree
=info-mode= provides commands for viewing the documentation tree. Many [[id:cf447557-1f87-4a07-916a-160cfd2310cf][GNU/Emacs]] packages will add their documentation here. It can be accessed with =<C-h i>=.
